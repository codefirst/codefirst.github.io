AsakusaSatellite

AsakusaSatellite とは

AsakusaSatellite は、開発者向けのリアルタイムチャットアプリケーションです。開発者同士のコミュニケーション品質を向上し、ソフトウェア開発にリズムをもたらします。

特徴

  • リアルタイム
  • ドラッグ＆ドロップによるファイルアップロード
  • Redmine のチケットリンク
  • プラグインによる拡張

動作環境

  • Ruby 1.8.7
  • Ruby on Rails 3.0.3
  • Google Chrome 9 or later (クライアント)

インストール

以下のコマンドを実行したあと http://localhost:3000/ にアクセスしてください。

$ git clone https://codefirst@github.com/codefirst/AsakusaSatellite.git
$ cd AsakusaSatellite
$ cp config/filter.yml.example config/filter.yml
$ cp config/websocket.yml.example config/websocket.yml
$ cp config/settings.yml.example config/filter.yml
$ bundle install --path vendor/bundle
$ rake groonga:migrate
$ ruby websocket/server &
$ rails server
